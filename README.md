# pokedexcli

Poke-dex via the CLI

Following along with [Boot Dev](https://youtu.be/8yrmAGcCnKg?si=qpeO5-kzG52h5cW3)

Start the [binary](./bin/pokedexcli) (MacOS).
Enter `help`

Build with `go build -o bin/pokedexcli cmd/pokedex/main.go`
